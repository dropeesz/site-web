## 📌 Sobre o Projeto

Portfólio pessoal desenvolvido para apresentar minha trajetória como desenvolvedor, projetos práticos, stack de tecnologias e serviços oferecidos.

O site conta com design moderno em tema escuro, terminal animado no hero, navbar responsiva com Bootstrap e seções bem estruturadas para transmitir identidade profissional.

---

## 🚀 Seções

* 🖥️ **Hero** — Apresentação com terminal animado e cursor piscante
* 👤 **Sobre** — Bio, foto e informações de formação
* 🛠️ **Tech** — Stack e ferramentas utilizadas
* 📂 **Projetos** — Repositórios públicos com descrição e links
* 🎨 **Serviços** — Design, Social Media, Sites e Sistemas
* 🎓 **Certificações** — Cursos e formações concluídas
* 📬 **Contato** — Links para LinkedIn, WhatsApp, GitHub e mais

## 🔗 Acesso

### Localmente

```bash
# Clone o repositório
git clone https://github.com/dropeesz/site-web.git

# Entre na pasta
cd site-web
```

Abra o arquivo `index.html` no navegador ou use a extensão **Live Server** no VS Code.

### Via GitHub Pages

```text
https://dropeesz.github.io/site-web/
```

> Para ativar: **Settings → Pages → Source: main / (root) → Save**
